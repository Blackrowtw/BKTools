# BKTools

[![License](https://img.shields.io/github/license/Blackrowtw/BKTools.svg)](http://www.gnu.org/licenses/lgpl-3.0.html)

A Minecraft Fabric mod built on Fallen_Breath's fabric-mod-template.
Uses the malilib library by masa and sakura-ryoko.

## Features

- Configurable command binding with loop modes
- Custom GUI widgets based on malilib
- And more...
